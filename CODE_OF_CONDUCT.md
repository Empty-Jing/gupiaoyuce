# 行为准则

为了让社区协作保持开放、友好与高效，项目参与者默认遵循以下原则：

## 我们鼓励的行为

- 尊重不同背景、经验与观点
- 基于事实讨论技术问题
- 接受建设性反馈，并愿意持续改进
- 在提问、评审和讨论时保持礼貌与克制

## 不被接受的行为

- 人身攻击、歧视、骚扰或羞辱性表达
- 持续发布与项目无关的挑衅内容
- 泄露他人隐私、凭据或未公开的安全信息
- 任何影响社区成员安全感的行为

## 适用范围

本准则适用于项目仓库中的 Issue、Pull Request、讨论区以及其他与本项目相关的公开协作场景。

## 处理方式

维护者有权删除不当内容、关闭相关讨论，必要时限制账号参与项目协作。

如果你遇到不当行为，请私下联系维护者处理。
